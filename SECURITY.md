# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in `cf-ready`, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities
2. Email security concerns to the maintainers via GitHub private security advisories on the repository

## Scope

- The `cf-ready` CLI tool itself
- False positive/negative handling in secret detection

Out of scope: vulnerabilities in scanned target applications (those should be reported to the application owner).
