@orangecloud/cf-readyMIT

Cloudflare Go-Live Readiness

Move from finished code to production-ready Cloudflare deployment with migration checks, security scanning, AI readiness, SEO readiness, and go-live reporting.

Quick start

npx @orangecloud/cf-ready scan

Install

npm install -g @orangecloud/cf-ready
cf-ready scan

Commands

  • cf-ready scan — full readiness scan
  • cf-ready inspect — repository detection
  • cf-ready migration-plan — Cloudflare migration plan
  • cf-ready security-scan — security checks + SARIF
  • cf-ready ai-ready — AI readiness checks
  • cf-ready seo-ready — SEO readiness checks
  • cf-ready fix --ai-readiness — generate AI assets
  • cf-ready fix --seo — generate SEO suggestions
  • cf-ready deploy-check — deployment readiness
  • cf-ready smoke-test --url <url> — live HTTP checks

Safety

Read-only by default. Does not deploy or migrate automatically. Only modifies files when you run explicit fix commands.
