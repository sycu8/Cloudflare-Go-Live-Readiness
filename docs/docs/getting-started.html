Getting started

CF Ready (cf-ready) is an open-source CLI for Cloudflare go-live readiness checks.

Quick scan

npx @orangecloud/cf-ready scan

Requirements

  • Node.js 18 or later
  • A project directory (local path, ZIP upload, or GitHub import in Web Agent)

Install globally

npm install -g @orangecloud/cf-ready
cf-ready scan --cwd ./my-app

Web Agent

No local install needed: open the Web Agent, upload source or import a public GitHub repo, then run scan.

Output

Scan generates up to 9 report files in your configured output directory (default: project root), including markdown report, JSON, SARIF, and migration plan.
