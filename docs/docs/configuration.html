Configuration

Optional cf-ready.config.json in your project root. Override with --config.

{
  "projectName": "my-app",
  "framework": "nextjs",
  "deploymentTarget": "cloudflare-pages",
  "outputDir": ".",
  "ai": {
    "workerUrl": "https://your-worker.workers.dev",
    "model": "openai/gpt-4o-mini"
  }
}

See examples/nextjs/cf-ready.config.json in the repository.
