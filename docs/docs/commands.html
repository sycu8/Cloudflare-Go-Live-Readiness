Commands

Global flags: --cwd, --config, --json, --verbose, --no-color

Core

  • scan — full readiness scan + all reports
  • inspect — detect framework, package manager, deployment target
  • report — regenerate reports from a fresh scan

Category scans

  • migration-plan — Cloudflare migration plan markdown
  • security-scan — security checks + SARIF
  • ai-ready — AI readiness (robots.txt, llms.txt, OpenAPI, MCP)
  • seo-ready — SEO readiness
  • deploy-check — deployment + migration + security subset

Actions

  • fix --ai-readiness / fix --seo — generate assets (explicit flags only)
  • ai-optimize — Workers AI refactor suggestions
  • smoke-test --url <url> — HTTP checks against live URL
