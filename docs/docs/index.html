CF Ready documentation

CF Ready (cf-ready) is an open-source CLI that helps developers check whether a website or application is ready to go live on Cloudflare.

Quick start

npx @orangecloud/cf-ready scan

See Getting started for install options and Web Agent for browser-based scans.

Topics

  • Migration readiness — runtime blockers, Next.js, Vite, Node.js
  • Security readiness — secrets, CORS, dependencies
  • AI readiness — robots.txt, llms.txt, OpenAPI, MCP
  • SEO readiness — metadata, sitemap, structured data
  • Reports — migration plan, SARIF, go-live checklist
  • GitHub Action — PR readiness checks
