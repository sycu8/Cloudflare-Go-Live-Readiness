Readiness categories

Overall score is weighted across five categories. Blockers prevent a production-ready status.

Weights

  • Migration (35%) — framework compatibility, Node/edge patterns, Cloudflare blockers
  • Security (30%) — headers, secrets, npm audit, CORS, env documentation
  • AI readiness (20%) — robots.txt, llms.txt, OpenAPI, MCP server card
  • SEO (10%) — metadata, sitemap, structured data
  • Deployment (5%) — wrangler, CI, scripts, rollback plan

Severity levels

Blocker must be resolved before production ready.

High Medium Low — prioritized recommendations.
