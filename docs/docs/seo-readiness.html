SEO readiness before production

Technical SEO checks for metadata, crawlability, and social previews.

Checks include

  • Title, description, and canonical URLs
  • Open Graph and Twitter card metadata
  • Structured data (JSON-LD) presence
  • Heading hierarchy and image alt text
  • robots.txt and sitemap.xml

Commands

cf-ready seo-ready
cf-ready fix --seo
